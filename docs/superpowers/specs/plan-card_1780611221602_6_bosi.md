## Architecture Decisions

**1. Session State & Caching Strategy**
- Use `st.session_state` to persist form inputs (URLs, selected filters) across reruns without storing sensitive auth data in state.
- Cache database queries with `@st.cache_data` (10-minute TTL) to avoid repeated SQLite reads. Invalidate on manual profile insertion.
- Session.json validation cached in `st.session_state` with manual refresh button—no polling loop to avoid blocking.

**2. Sidebar vs. Main Layout**
- Sidebar contains system operations (validation, auth trigger, file size check); main area contains data input and output.
- Rationale: Keeps ops separate from analytics. User can access session control without losing context in data view.

**3. Multi-Tab Design**
- Tab 1: "Input & Auth" — bulk URL textarea, submit button, interactive auth routine trigger.
- Tab 2: "Analytics" — metrics cards (total profiles, new today, avg experience years), filtered dataframe, export button.
- Rationale: Logical separation of write vs. read operations reduces UI clutter.

**4. DataFrame Display & Export**
- Use `st.dataframe()` with `use_container_width=True` for responsive display. Pre-calculate JSON fields (experience, education) as serialized previews in derived column.
- Export via `st.download_button` with `BytesIO` buffer. Support both CSV (default) and Excel (XLSX via openpyxl).
- Rationale: Responsive table reflows on mobile. BytesIO avoids temp file I/O on disk.

**5. Real-Time Validation Display**
- Display validation state as colored badges (🟢 Valid, 🔴 Missing, 🟡 Expired) refreshed on button click.
- Validation logic: check file exists, parse JSON structure (cookies + origins keys), attempt live test load (async).
- Rationale: Avoids blocking app with async operations; user controls validation trigger timing.

**6. Error Handling & Logging**
- Catch exceptions in session validation and display user-friendly messages in `st.error()` blocks.
- Log to console via standard logging module (not st.write()) to avoid polluting UI.
- Never expose stack traces to user—summarize as "Session invalid - missing required keys" or "Database unavailable".

```python
"""Helper functions for Streamlit UI components."""

import json
import logging
from pathlib import Path
from typing import Tuple, Optional
import asyncio
from datetime import datetime

logger = logging.getLogger(__name__)


def validate_session_file(session_file: str = "session.json") -> Tuple[bool, str, Optional[str]]:
    """
    Validate session.json file structure and content.
    
    Returns:
        Tuple of (is_valid: bool, status_message: str, expiry_info: Optional[str])
    """
    session_path = Path(session_file)
    
    # Check file exists
    if not session_path.exists():
        return False, "❌ Session file not found", None
    
    # Check file size
    file_size = session_path.stat().st_size
    if file_size == 0:
        return False, "❌ Session file empty", None
    
    # Validate JSON structure
    try:
        with open(session_path) as f:
            data = json.load(f)
    except json.JSONDecodeError as e:
        return False, f"❌ Invalid JSON: {str(e)[:50]}", None
    
    # Check required keys
    required_keys = ["cookies", "origins"]
    if not all(key in data for key in required_keys):
        missing = [k for k in required_keys if k not in data]
        return False, f"❌ Missing keys: {', '.join(missing)}", None
    
    # Check cookies present
    if not data.get("cookies"):
        return False, "❌ No authentication cookies found", None
    
    # Success - calculate cookie expiry if available
    expiry_info = None
    if data.get("cookies"):
        # Get earliest expiration from cookies
        expirations = []
        for cookie in data["cookies"]:
            if "expires" in cookie:
                expirations.append(cookie["expires"])
        if expirations:
            earliest = min(expirations)
            expiry_info = f"Expires: {datetime.fromtimestamp(earliest).strftime('%Y-%m-%d %H:%M:%S')}"
    
    return True, "🟢 Session valid & active", expiry_info


def calculate_profile_metrics(df: pd.DataFrame) -> dict:
    """
    Calculate dashboard metrics from profile dataframe.
    
    Returns:
        Dict with keys: total_count, new_today, avg_experience_years
    """
    import pandas as pd
    from datetime import datetime, timedelta
    
    if df.empty:
        return {
            "total_count": 0,
            "new_today": 0,
            "avg_experience_years": 0
        }
    
    # Total profiles
    total_count = len(df)
    
    # New profiles today
    df["collected_at"] = pd.to_datetime(df["collected_at"])
    today = datetime.now().replace(hour=0, minute=0, second=0, microsecond=0)
    new_today = len(df[df["collected_at"] >= today])
    
    # Average experience years (parsed from experience_json)
    avg_exp = 0
    try:
        exp_years = []
        for exp_json_str in df["experience_json"].dropna():
            if exp_json_str:
                import json
                exp_data = json.loads(exp_json_str)
                if isinstance(exp_data, list) and len(exp_data) > 0:
                    exp_years.append(len(exp_data))
        if exp_years:
            avg_exp = sum(exp_years) / len(exp_years)
    except Exception as e:
        logger.warning(f"Failed to calculate avg experience: {e}")
    
    return {
        "total_count": total_count,
        "new_today": new_today,
        "avg_experience_years": round(avg_exp, 1)
    }


def display_validation_sidebar(session_file: str = "session.json") -> None:
    """Render sidebar validation UI."""
    import streamlit as st
    
    st.sidebar.markdown("## 🔐 System Operations")
    
    col1, col2 = st.sidebar.columns([3, 1])
    with col1:
        st.sidebar.markdown("**Session Status**")
    with col2:
        if st.sidebar.button("🔄 Refresh", key="refresh_validation"):
            st.session_state.session_validated = False
    
    # Validate on demand or cache if recently validated
    if (not st.session_state.session_validated or 
        st.session_state.validation_timestamp is None):
        is_valid, status_msg, expiry_info = validate_session_file(session_file)
        st.session_state.session_validated = is_valid
        st.session_state.validation_status = status_msg
        st.session_state.validation_expiry = expiry_info
    
    # Display status badge
    status_container = st.sidebar.container()
    status_container.markdown(st.session_state.validation_status)
    
    if st.session_state.validation_expiry:
        st.sidebar.caption(st.session_state.validation_expiry)
    
    # File size indicator
    session_path = Path(session_file)
    if session_path.exists():
        size_kb = session_path.stat().st_size / 1024
        st.sidebar.caption(f"📁 {size_kb:.1f} KB")
```

```python
def prepare_profiles_dataframe(df: pd.DataFrame) -> pd.DataFrame:
    """
    Prepare dataframe for display: parse JSON fields, derive preview columns.
    
    Applies pandas-pro best practices:
    - Vectorized operations instead of iteration
    - Explicit type conversion
    - Memory-efficient display subset
    """
    import json
    import pandas as pd
    
    if df.empty:
        return df
    
    # Make explicit copy to avoid SettingWithCopyWarning
    df_display = df.copy()
    
    # Convert collected_at to datetime
    df_display["collected_at"] = pd.to_datetime(df_display["collected_at"])
    
    # Parse experience_json: count jobs
    def count_jobs(exp_json_str):
        if pd.isna(exp_json_str) or not exp_json_str:
            return 0
        try:
            data = json.loads(exp_json_str)
            return len(data) if isinstance(data, list) else 0
        except:
            return 0
    
    # Parse education_json: count degrees
    def count_degrees(edu_json_str):
        if pd.isna(edu_json_str) or not edu_json_str:
            return 0
        try:
            data = json.loads(edu_json_str)
            return len(data) if isinstance(data, list) else 0
        except:
            return 0
    
    # Vectorized: extract parsed counts
    df_display["Jobs"] = df_display["experience_json"].apply(count_jobs)
    df_display["Degrees"] = df_display["education_json"].apply(count_degrees)
    
    # Select display columns in order
    display_cols = [
        "full_name",
        "headline",
        "location",
        "current_company",
        "Jobs",
        "Degrees",
        "collected_at"
    ]
    
    df_display = df_display[display_cols].copy()
    
    # Rename for display
    df_display.columns = [
        "Name",
        "Headline",
        "Location",
        "Company",
        "Jobs",
        "Degrees",
        "Collected"
    ]
    
    # Format timestamp
    df_display["Collected"] = df_display["Collected"].dt.strftime("%Y-%m-%d %H:%M")
    
    return df_display.sort_values("Collected", ascending=False)
```

```python
"""Asset export helpers for dataframe to CSV/Excel conversion."""

import io
import json
import logging
from typing import Optional
import pandas as pd

logger = logging.getLogger(__name__)


def export_profiles_to_csv(df: pd.DataFrame) -> bytes:
    """
    Export profile dataframe to CSV format.
    
    Args:
        df: Raw profile dataframe from database
    
    Returns:
        CSV bytes ready for download
    """
    if df.empty:
        return b""
    
    # Use prepared display dataframe
    from ui_helpers import prepare_profiles_dataframe
    df_display = prepare_profiles_dataframe(df)
    
    # Convert to CSV with UTF-8 encoding
    csv_buffer = io.StringIO()
    df_display.to_csv(csv_buffer, index=False, encoding='utf-8')
    
    return csv_buffer.getvalue().encode('utf-8')


def export_profiles_to_excel(df: pd.DataFrame) -> bytes:
    """
    Export profile dataframe to Excel (XLSX) format.
    
    Args:
        df: Raw profile dataframe from database
    
    Returns:
        Excel bytes ready for download
    """
    if df.empty:
        return b""
    
    # Use prepared display dataframe
    from ui_helpers import prepare_profiles_dataframe
    df_display = prepare_profiles_dataframe(df)
    
    # Create BytesIO buffer
    excel_buffer = io.BytesIO()
    
    # Write to Excel with basic formatting
    with pd.ExcelWriter(excel_buffer, engine='openpyxl') as writer:
        df_display.to_excel(
            writer,
            sheet_name='Profiles',
            index=False,
            freeze_panes=(1, 0)
        )
        
        # Auto-adjust column widths
        worksheet = writer.sheets['Profiles']
        for idx, col in enumerate(df_display.columns, 1):
            max_length = max(
                df_display[col].astype(str).str.len().max(),
                len(col)
            ) + 2
            worksheet.column_dimensions[chr(64 + idx)].width = min(max_length, 50)
    
    excel_buffer.seek(0)
    return excel_buffer.getvalue()


def generate_export_filename(extension: str) -> str:
    """Generate timestamped export filename."""
    from datetime import datetime
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    return f"linkedin_profiles_export_{timestamp}.{extension}"
```

```python
class TestSessionValidation:
    """Test session.json validation logic."""
    
    def test_validate_session_file_not_found(self):
        """Missing session file returns invalid."""
        is_valid, status, expiry = validate_session_file("/nonexistent/path.json")
        assert is_valid is False
        assert "not found" in status.lower()
    
    def test_validate_session_file_empty(self):
        """Empty session file returns invalid."""
        with tempfile.NamedTemporaryFile(mode='w', suffix='.json', delete=False) as f:
            temp_path = f.name
        
        try:
            is_valid, status, expiry = validate_session_file(temp_path)
            assert is_valid is False
            assert "empty" in status.lower()
        finally:
            Path(temp_path).unlink()
    
    def test_validate_session_file_invalid_json(self):
        """Malformed JSON returns invalid."""
        with tempfile.NamedTemporaryFile(mode='w', suffix='.json', delete=False) as f:
            f.write("{ invalid json")
            temp_path = f.name
        
        try:
            is_valid, status, expiry = validate_session_file(temp_path)
            assert is_valid is False
            assert "json" in status.lower()
        finally:
            Path(temp_path).unlink()
    
    def test_validate_session_file_missing_keys(self):
        """Session missing required keys returns invalid."""
        with tempfile.NamedTemporaryFile(mode='w', suffix='.json', delete=False) as f:
            json.dump({"cookies": []}, f)
            temp_path = f.name
        
        try:
            is_valid, status, expiry = validate_session_file(temp_path)
            assert is_valid is False
            assert "origins" in status
        finally:
            Path(temp_path).unlink()
    
    def test_validate_session_file_valid(self):
        """Valid session file returns True."""
        with tempfile.NamedTemporaryFile(mode='w', suffix='.json', delete=False) as f:
            json.dump({
                "cookies": [{"name": "li_at", "value": "token123"}],
                "origins": [{"origin": "https://linkedin.com"}]
            }, f)
            temp_path = f.name
        
        try:
            is_valid, status, expiry = validate_session_file(temp_path)
            assert is_valid is True
            assert "valid" in status.lower()
        finally:
            Path(temp_path).unlink()


class TestMetricsCalculation:
    """Test profile metrics calculation."""
    
    def test_metrics_empty_dataframe(self):
        """Empty dataframe returns zero metrics."""
        df = pd.DataFrame()
        metrics = calculate_profile_metrics(df)
        
        assert metrics["total_count"] == 0
        assert metrics["new_today"] == 0
        assert metrics["avg_experience_years"] == 0
    
    def test_metrics_total_count(self):
        """Total count reflects dataframe length."""
        df = pd.DataFrame({
            "full_name": ["Alice", "Bob"],
            "collected_at": [datetime.now(), datetime.now()]
        })
        metrics = calculate_profile_metrics(df)
        
        assert metrics["total_count"] == 2
    
    def test_metrics_new_today(self):
        """New today includes only recent profiles."""
        df = pd.DataFrame({
            "full_name": ["Alice", "Bob", "Charlie"],
            "collected_at": [
                datetime.now(),
                datetime.now(),
                "2020-01-01"
            ]
        })
        metrics = calculate_profile_metrics(df)
        
        assert metrics["new_today"] == 2
    
    def test_metrics_average_experience(self):
        """Average experience calculated from experience_json."""
        df = pd.DataFrame({
            "full_name": ["Alice", "Bob"],
            "collected_at": [datetime.now(), datetime.now()],
            "experience_json": [

                json.dumps([{"title": "PM"}])  # 1 job
            ]
        })
        metrics = calculate_profile_metrics(df)
        
        assert metrics["avg_experience_years"] == 1.5
```

## State Management Strategy

**Session State Variables:**
- `session_validated` (bool) — Cache validation result across reruns
- `validation_timestamp` (datetime) — Track when validation last ran
- `validation_status` (str) — Cached status message
- `validation_expiry` (str) — Cached expiry info from cookies
- `bulk_urls` (str) — Persist textarea input across tab switches
- `last_queue_time` (datetime) — Track last queue submission

**Database Caching:**
- `@st.cache_data(ttl=600)` on `load_profile_data()` — Refresh every 10 minutes
- Cache invalidated when user clicks "Queue for Scraping" (manual invalidation)

**Reasoning:**
- Session state prevents refetch of form inputs on every rerun
- 10-minute cache TTL balances freshness vs. performance
- Manual refresh button gives user control over validation timing (no blocking async)

## Accessibility Considerations

**WCAG 2.1 AA Compliance:**

1. **Color & Contrast:**
   - Validation badges use both color AND text (🟢 Valid, 🔴 Missing)
   - Metric cards use sufficient contrast (Streamlit default theme)

2. **Navigation:**
   - Tab layout uses semantic structure (can keyboard navigate)
   - Form labels are descriptive ("Paste LinkedIn profile URLs...")
   - Sidebar refresh button is keyboard accessible (button element)

3. **Data Tables:**
   - Dataframe has frozen header row (Excel export, accessibility)
   - Column names are descriptive ("Jobs" instead of "job_count")
   - Responsive: table scrolls horizontally on mobile

4. **Form Controls:**
   - Text area has explicit placeholder text
   - Multiselect has clear label ("Filter by Company")
   - Numeric input has min/max constraints

5. **Error Messages:**
   - Warnings use `st.warning()` (colored, prominent)
   - Errors use `st.error()` (red background)
   - Info messages use `st.info()` (blue background)
   - All messages include emoji for non-color-dependent indication

**Not Addressed (Out of Scope):**
- Screen reader optimization (requires custom HTML components)
- ARIA labels (would require custom Streamlit component)
- Dark mode testing (Streamlit handles via theme config)

## Risks and Edge Cases

| Risk | Likelihood | Impact | Mitigation |
|------|------------|--------|-----------|
| Session token expires during session | Medium | Validation shows ❌ but scraper still works (until token used) | Add "Refresh Session" link in sidebar pointing to Launch button |
| Large dataset (10k+ profiles) causes lag in dataframe | Medium | UI becomes slow on tab switch | Implement row limit (default 50), cache with longer TTL, lazy-load dataframe |
| User pasts non-LinkedIn URLs | High | Queue button filters invalid URLs silently | Add warning highlighting invalid URLs, show count of queued vs. total |
| Excel export fails if openpyxl not installed | Low | Download button throws error | Add openpyxl to requirements.txt; test export before showing button |
| Streamlit session resets (browser refresh) | High | Form inputs lost | Use st.session_state to persist textarea across reruns (already done) |
| Database file locked by another process | Low | Query fails, dataframe empty | Catch exception in load_profile_data(), show error message |
| Mobile view: table too wide to scroll | Medium | Columns cut off | st.dataframe with use_container_width=True handles auto-scroll |
